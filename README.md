# Laveen Home Health Agency Website

A modern, premium, SEO-focused website for a home healthcare company built with React, Framer Motion, and Tailwind CSS.

## 🎨 Design Features

- **Premium Soft Luxury Aesthetic**: Clean, elegant design with warm colors and sophisticated typography
- **Smooth Animations**: Framer Motion animations throughout for a polished, professional feel
- **Responsive Design**: Mobile-first approach that looks beautiful on all devices
- **SEO Optimized**: Proper heading hierarchy, meta tags, and semantic HTML
- **Conversion Focused**: Strategic CTAs and clear user journeys

## 🎯 Brand Identity

**Color Palette:**
- Warm ivory/beige backgrounds
- Deep burgundy accents (#B84444)
- Charcoal text (#4A4A4A)
- Soft taupe secondary tones

**Typography:**
- Headlines: Playfair Display (elegant, distinctive)
- Subheadings: Cormorant Garamond (refined serif)
- Body: Inter (clean, readable sans-serif)

## 📄 Pages

1. **Home** - Hero, services overview, testimonials, CTAs
2. **About Us** - Mission, values, personalized approach
3. **Services** - Skilled and non-skilled services with detailed descriptions
4. **Who We Serve** - Target audiences with specific care needs
5. **Careers** - Team benefits and application form
6. **Contact** - Contact form, info, and map

## 🚀 Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Visit `http://localhost:3000` to view the website.

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## 🛠️ Technology Stack

- **React 18** - Modern UI library
- **Vite** - Fast build tool and dev server
- **React Router** - Client-side routing
- **Framer Motion** - Smooth animations
- **Tailwind CSS** - Utility-first styling
- **Lucide React** - Beautiful icons
- **React Helmet Async** - SEO meta tags
